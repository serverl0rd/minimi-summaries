# 📓 Minimi Daily Summaries

A personal activity log — automatically generated each morning from [Minimi](https://www.projectminimi.com/), capturing everything that happened the day before across messages, email, documents, browsing, and calendar.

---

## What's in here

Each file is named `YYYY-MM-DD.md` and covers one day. Inside, you'll find:

- **Messages & Chats** — key conversations across WhatsApp, iMessage, Slack, and more
- **Emails** — significant emails sent or received
- **Web & Research** — sites visited, topics explored
- **Documents & Work** — anything written, read, or worked on
- **Calendar & Meetings** — events attended and outcomes
- **Key Highlights** — the 3–5 things that actually mattered that day

---

## How it works

A scheduled Claude task runs every morning at **8:00 AM IST**. It:

1. Pulls the previous day's activity from Minimi (locally captured, end-to-end private)
2. Synthesizes it into a structured markdown summary
3. Commits and pushes the file to this repo
4. Sends a copy to Gmail as a daily digest

The summaries are generated by [Claude](https://claude.ai) using the Minimi MCP connector — no data leaves your machine except what you explicitly push here.
